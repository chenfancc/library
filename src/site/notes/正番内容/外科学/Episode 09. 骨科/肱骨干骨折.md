# 概述
+ 部位：外科颈下 1-2 cm～肱骨髁上 2 cm
+ 损伤机制：
	+ 直接暴力（横行骨折）
	+ 间接暴力（斜行、螺旋形骨折）
	+ 病理性骨折
# 合并损伤
+ 注意桡神经
# 临床表现及诊断
+ 外伤史
+ 疼痛、肿胀、畸形、反常活动等症状及体征常常可以出现
+ 临床检查应特别注意有无桡神经损伤的临床表现：垂腕、虎口区感觉减退、掌指关节不能背伸
+ X 线可以明确骨折类型、骨折的移位情况
# 治疗
+ 手法复位，外固定
+ 切开复位，内固定
	+ 手术指征：
# 并发症
+ 桡神经损伤
+ 骨折不愈合
+ 畸形愈合
+ 感染
+ 骨化性肌炎
